### 运营

```
收入=用户数*转化率*复购*客单
```

```
用户数=流量*转化率
```

```
流量=自然流量+购买流量+裂变
```

```
抖音流量=视频自然流量+购买流量+直播流量
```

```
优质的内容,理想情况

播放量/20=粉丝数
```

```
一般内容
播放量/50=粉丝数
```

```
购买流量/20=粉丝数
```

```
直播/20=粉丝数
```

运营如何做内容

1. 设定目标(北极星指数)

  做短视频之前，目标是什么，做一个活动，目标是什么

2. 拆分

  要想涨粉，通过自然流量来

3. 带入真实的数据

  学会读数据

4. 用基础数学搞定它

#### example

目标: 今晚要涨粉5000

拆分: 

自然流量/20 + 购买流量/20 + 直播/20 = 目标

真实数据

每50个看完视频的人，则有一个人关注
